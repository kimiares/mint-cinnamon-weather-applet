This applet uses OpenWeatherMap. To set your API key:

1) GUI: Right-click the applet → Configure → enter API key in "OpenWeatherMap API key".
2) Manual: Edit the GSettings key 'api-key' for schema org.cinnamon.applets.mint-weather, or run:
   gsettings set org.cinnamon.applets.mint-weather api-key 'YOUR_KEY'

Do not commit the API key to version control.
